Frequently Asked Questions

What HTML entities does it decode?

It decodes all standard HTML entities including named entities (&, <, >, ",  ), decimal entities (&), and hexadecimal entities (&).

Can it decode all special HTML characters?

Yes! The decoder handles all 2,000+ named HTML entities, as well as any valid decimal or hexadecimal numeric entity reference.

Why would I need to decode HTML entities?

Common reasons include extracting clean text from HTML source code, processing scraped web content, fixing double-encoded text, and converting HTML email content to plain text.

What is double encoding?

Double encoding happens when already-encoded text gets encoded again, producing things like &amp; instead of &. Our decoder converts one level of encoding at a time.
