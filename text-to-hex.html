Frequently Asked Questions

What is hexadecimal?

Hexadecimal (hex) is a base-16 number system using digits 0-9 and letters A-F. Each hex digit represents 4 bits, making it a compact way to represent binary data. Two hex digits represent one byte.

How is text converted to hex?

Each character's ASCII or Unicode code point is converted to hexadecimal. For example, 'A' is ASCII 65, which is 41 in hex. 'Hello' becomes 48 65 6C 6C 6F.

Is this the same as HTML color codes?

HTML color codes use hexadecimal notation (#RRGGBB), but they represent color values, not text. The conversion process is the same base-16 numbering system, though.

Can it handle non-English characters?

Yes! Non-ASCII characters are encoded using UTF-8, which may produce multiple hex bytes per character. For example, the euro sign (€) becomes E2 82 AC in UTF-8 hex.
