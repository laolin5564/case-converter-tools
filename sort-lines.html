Sort text lines in alphabetical order (A-Z or Z-A)

What is Sort Lines?

Our Sort Lines tool arranges lines of text in alphabetical, numerical, or reverse order. It's perfect for organizing lists, sorting data, cleaning up text files, and preparing content for further processing.

The tool offers multiple sorting options: ascending (A-Z), descending (Z-A), and reverse order. You can also remove duplicate lines while sorting, combining two common operations in one step.

Whether you're sorting a list of names, organizing CSV rows, or alphabetizing a glossary, this tool handles it instantly with no size limits.

How to Use Sort Lines

  1. Paste your text with one item per line
  2. Choose sort order: ascending, descending, or reverse
  3. Optionally enable duplicate removal
  4. Click Sort and copy the organized result

Frequently Asked Questions

Does it sort numerically or alphabetically?

By default, sorting is alphabetical (lexicographic). Numbers are sorted as text, so '10' comes before '2'. For true numerical sorting, the tool may offer a numeric sort option.

Can it sort case-insensitively?

The sort may be case-sensitive by default (uppercase before lowercase). Check if a case-insensitive option is available for mixed-case sorting.

Does it handle Unicode characters?

Yes, the sorter handles Unicode text and sorts according to Unicode code points. This works well for most languages, though locale-specific sorting rules may vary.

Can I remove duplicates while sorting?

Yes! The tool can remove duplicate lines as part of the sorting process, giving you a clean, sorted list with no repetitions.
