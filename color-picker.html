Frequently Asked Questions

What color formats are supported?

The color picker supports HEX (#FF0000), RGB (rgb(255,0,0)), HSL (hsl(0,100%,50%)), and may also show HSV and CMYK values.

Can I enter a color code to see it?

Yes! Type any valid color code (HEX, RGB, or HSL) and the picker will show you the color visually and convert it to all other formats.

What's the difference between HEX and RGB?

They represent the same colors differently. HEX uses hexadecimal notation (#RRGGBB), while RGB uses decimal values (0-255 for each channel). Both are widely used in CSS.

Can I save or export colors?

You can copy any color code format to your clipboard. For saving color palettes, copy the codes you need and store them in your project's design system or CSS files.
